This app saves to this device only until you connect Firebase. To share live data with everyone in the family:

  1. Go to console.firebase.google.com and create a free project.
  2. Open Build → Firestore Database and create a database in test mode.
  3. Go to Project settings → Your apps → add a Web app and copy the firebaseConfig shown.
  4. Open this HTML file in a text editor, find FIREBASE_CONFIG near the top of the <script>, and replace the placeholder values with yours.
  5. Save, then send the same file to everyone. All copies read and write the same shared data automatically.
